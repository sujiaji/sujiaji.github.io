# Jiaji's Homepage

This page is modified based on **Minimal**:

*Minimal is a Jekyll theme for GitHub Pages. You can [preview the theme to see what it looks like](http://pages-themes.github.io/minimal), or even [use it today](https://github.com/pages-themes/minimal?tab=readme-ov-file#usage).*


If you want to start with **cloning my repository**, remember to revise:
- _config.yml
- index.md
- pages/
- images/profile.png
- files/CV.pdf

**Meanwhile** in the footers of *_layouts/default.html* and *_layouts/default.html*, there is a widget visit tracker from [https://clustrmaps.com/](https://clustrmaps.com/) need to be replaced/removed accordingly.

To change the content in navigation bar, revise *_data/navigation.yml*.

Hope this repository is a good start for you.
